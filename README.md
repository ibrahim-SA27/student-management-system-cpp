# 🎓 Student Management System (C++)

## 📌 Project Description
This is a console-based Student Management System built using C++.  
It uses file handling to store student data permanently and provides a simple menu-driven interface.

## 🚀 Features
- Add Student Record
- Display All Students
- Search Student by ID
- Update Student Details
- Delete Student Record
- Persistent storage using file handling

## 🛠️ Technologies Used
- C++
- File Handling (Binary Files)
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Copy the code into any C++ compiler (like Cxxdroid)
2. Run the program
3. Use menu options to manage student records

## 📂 File Used
- `students.dat` → stores all student data

## 🎯 Learning Outcomes
- Understanding file handling in C++
- Implementing CRUD operations
- Designing menu-driven programs
